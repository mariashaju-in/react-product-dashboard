# React Product Dashboard

This is a simple frontend dashboard built using React to display product data from a backend API.

The goal of this project was to understand how frontend applications interact with backend services and how data can be presented in a user-friendly way.

---

## What this project does

- Fetches product data from backend API  
- Displays product list  
- Search products by name  
- Shows summary info like total products and low stock items  

---

## Tech Stack

- React (Vite)
- JavaScript
- Fetch API

---

## Why I built this

After building a backend Product Catalog API, I wanted to understand how frontend and backend connect in a real application.  
This project helped me learn how to fetch data, manage state, and build reusable UI components.
